# GitHub Pages Website

This project is a website written using HTML5/CSS3 best practices in accessibility and responsive design. It utilizes modern javascript without additional libraries or frameworks to create fully functional games and dynamic navigation.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

```
A web browser
```

### Installing

You will need to have a modern web browser installed on your machine.


```
Firefox, Safari, Edge, etc...
```

Feel free to visit the [website](https://mossandocean.ca/) to see it in action.
